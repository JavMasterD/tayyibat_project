# 🌿 نظام الطيبات - تطبيق ويب

تطبيق ويب لتحليل الأطعمة وفق **نظام الطيبات للدكتور ضياء العوضي**،
مع متابعة قياسات ضغط الدم والسكر.

---

## 📁 هيكل المشروع

```
tayyibat/
├── index.html   → الصفحة الرئيسية
├── style.css    → التصميم والألوان
├── data.js      → قاعدة بيانات الأطعمة (70+ طعام)
├── app.js       → المنطق البرمجي الكامل
└── README.md    → هذا الملف
```

---

## 🚀 طريقة التشغيل

افتح ملف `index.html` مباشرة في المتصفح.

> **ملاحظة:** ميزة تحليل الذكاء الاصطناعي تعمل فقط عند تشغيل التطبيق داخل منصة Claude.ai

---

## ✨ المميزات

### 🔍 تحليل الطعام
- بحث ذكي مع اقتراحات فورية
- قاعدة بيانات تضم **70+ طعام** مصنّف
- تحليل منطقي لكل طعام (سبب القبول أو الرفض)
- تصفح حسب الفئة (نشويات، بروتينات، خضروات...)
- تحليل بالذكاء الاصطناعي للأطعمة غير الموجودة

### 📊 متابعة الصحة
- تسجيل قياسات ضغط الدم
- تسجيل قياسات سكر الدم
- تصنيف تلقائي (طبيعي / مرتفع / منخفض)
- مخطط بياني لسكر الدم
- سجل كامل مع تاريخ ووقت كل قياس
- حفظ تلقائي في المتصفح
